# Aegean Week

Mediterranean-style weekly diet planner focused on daily potassium and magnesium coverage.

## Run

1. `cd "C:\Users\JAson\Documents\codex\New project\apps\aegean-week"`
2. `node server.js`
3. Open `http://localhost:30005`

## Files

- [`index.html`](./index.html) is the planner entry page.
- [`diet.css`](./diet.css) contains the planner styling.
- [`src/dietPlannerApp.js`](./src/dietPlannerApp.js) renders the UI.
- [`src/dietPlannerData.js`](./src/dietPlannerData.js) contains meals, presets, and weekly plan data.
- [`src/dietPlannerLogic.js`](./src/dietPlannerLogic.js) contains nutrient summaries and helper logic.
